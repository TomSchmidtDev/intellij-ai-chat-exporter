Export your GitHub Copilot and Claude Code chat history to Markdown or styled HTML, right from your IDE.

GitHub Copilot
  • Browse all Copilot chat sessions: inline chat, agent mode, and edit mode
  • Filter sessions by IDE (current IDE or all JetBrains IDEs)
  • Select individual messages or entire sessions for export
Claude Code
  • Browse all Claude Code sessions from ~/.claude/projects/ — CLI, Claude Desktop, and JetBrains IDE sessions included
  • Filter by project directory
  • Toggle tool calls and thinking blocks on/off (hidden by default)
Export
  • Export to .md (Markdown) or .html (dark-themed, syntax-highlighted)
  • 10 built-in color profiles for HTML export — fully customizable
  • Works on Linux, macOS, and Windows
  • Read-only access — your data is never modified
Requirements
  • For Copilot export: GitHub Copilot must be installed with at least one chat session recorded
  • For Claude Code export: Claude Code must be installed with at least one session in ~/.claude/projects/
