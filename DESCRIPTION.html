Export your GitHub Copilot chat history to Markdown or styled HTML, right from your IDE.

Features
  • Browse all Copilot chat sessions: inline chat, agent mode, and edit mode
  • Select individual messages or entire sessions for export
  • Export to .md (Markdown) or .html (dark-themed, syntax-highlighted)
  • 10 built-in color profiles for HTML export — fully customizable
  • Works on Linux, macOS, and Windows
  • Reads the Copilot database read-only — your data is never modified
Requirements
  • GitHub Copilot must be installed and have been used in your JetBrains IDE at least once
